    @function hf-selector($type, $names) { ... }

    Description

    Creates a hydro selector using the set prefixes and a multidimensional list of names.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value

    		String or List—none

    		Booleantrue

    		Stringnull
